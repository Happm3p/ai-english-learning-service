# LP設計書

## 1. ページ概要
- **目的**: サービスの価値を瞬時に伝え、ユーザー登録（利用開始）へ誘導する。
- **ターゲット**: 英語学習中の社会人・学生（ペルソナA, B）
- **トーン＆マナー**: 知的、信頼感、先進的、シンプル。

## 2. 構成要素（セクション）

### 2.1 ヘッダー
- **ロゴ**: 左寄せ。サービス名「AI English Corrector」
- **ナビゲーション**: なし（LPとしての離脱防止のためシンプルに）
- **CTA**: 右寄せ。「ログイン / 登録」ボタン

### 2.2 ヒーローセクション (First View)
- **キャッチコピー**: "AIで、英語はもっと自由になる。"
- **サブコピー**: "Claude Haiku 4.5搭載。あなたの英文を瞬時に添削し、ネイティブのような表現力を。"
- **CTAボタン**: "無料で始める"（目立つ色で）
- **ビジュアル**: 英文がAIによって洗練されていくイメージのアニメーションやイラスト。

### 2.3 特徴セクション (Features)
- **Feature 1: 爆速フィードバック**
  - アイコン: 雷 (FlashOn)
  - 説明: 入力から数秒で添削完了。思考を止めずに学習を継続できます。
- **Feature 2: 深い洞察と解説**
  - アイコン: 分析 (Analytics/Psychology)
  - 説明: 単なる修正だけでなく、「なぜ」修正が必要なのかを丁寧に解説します。
- **Feature 3: 成長の可視化**
  - アイコン: グラフ (TrendingUp)
  - 説明: 学習履歴を自動保存。過去のミスを振り返り、確実なスキルアップをサポート。

### 2.4 使い方セクション (How it works)
- **Step 1**: 英文を入力
- **Step 2**: AIが即座に添削・解説
- **Step 3**: 履歴で復習・定着

### 2.5 ボトムCTA
- **メッセージ**: "まずは、あなたの英語をチェックしてみましょう。"
- **CTAボタン**: "今すぐ無料で試す"

### 2.6 フッター
- 著作権表示
- 利用規約、プライバシーポリシーへのリンク（仮）

## 3. デザイン・技術要件
- **フレームワーク**: Next.js 15 (App Router)
- **UIライブラリ**: MUI (Material UI) v6
- **レスポンシブ**: モバイル、タブレット、デスクトップで見やすく調整。
- **カラーパレット**:
  - Primary: 知的なブルー (#1976d2)
  - Secondary: アクセントのオレンジ (#ed6c02)
  - Background: ダークモード対応（白 / ダークグレー）
- **ダークモード**: ユーザーのシステム設定またはトグルで切り替え可能にする（今回はシステム設定依存またはデフォルトダーク寄りでも可）。

## 4. 実装ファイル
- `src/app/page.tsx`: LPのメインコンポーネント
- `src/app/layout.tsx`: 全体のレイアウト（MUIプロバイダー設定含む）
