## 2. レスポンシブ確認 (Responsive Check)

### 2.1 PCサイズ (幅 1024px以上)
- [ ] ヘッダーのメニューやボタンが横並びで適切に配置されていること。
- [ ] 特徴セクションのカードが3列（横並び）で表示されていること。
- [ ] 使い方セクションのステップが3列（横並び）で表示されていること。

### 2.2 タブレットサイズ (幅 768px - 1023px)
- [ ] コンテンツが画面幅に収まり、横スクロールが発生していないこと。
- [ ] グリッドレイアウトが適切に折り返されているか、縮小されていること。

### 2.3 スマホサイズ (幅 767px以下)
- [ ] ヘッダーのレイアウトが崩れていないこと（必要に応じてハンバーガーメニュー化など、現状の実装に合わせて確認）。
- [ ] 特徴セクションのカードが1列（縦積み）になっていること。
- [ ] 使い方セクションのステップが1列（縦積み）になっていること。
- [ ] 文字サイズが小さすぎて読めない箇所がないこと。
